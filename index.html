Azure CLI Command

This app shows reference data refreshed monthly. For real-time, subscription-specific VM SKU availability including restrictions, run:

az vm list-skus --location newzealandnorth --resource-type virtualMachines -o table

Requires Azure CLI and at least Reader role on your subscription.
